# 🛡️ AI Sentinel
### *Expose the AI Behind Any File*

**AI Sentinel** is a full-stack, animated web application that detects AI-generated content
in documents, code files, images, and videos — giving you a precise confidence score,
model-style attribution (ChatGPT-like, Gemini-like, Claude-like), and highlighted passages.

[🌐 Live Demo](https://ai-sentinel-kislay.vercel.app) · [📡 API](https://MohammadSakibAhmad0874-ai-sentinel.hf.space/api/docs) · [🐛 Issues](https://github.com/MohammadSakibAhmad0874/AI-Sentinel/issues)

</div>

## 📖 What Is This Project?

AI Sentinel is an **AI Content Detection Platform** built for students, educators, journalists, and content creators who want to know whether a piece of writing, image, or video was generated by an AI model.

Unlike simple text classifiers, AI Sentinel:
- Supports **15+ file formats** — not just plain text
- Scans **images** for AI generation signals (EXIF metadata + visual heuristics)
- Analyses **videos** frame-by-frame with a 2-minute / 100 MB safety cap
- Provides **model-style attribution** — estimates how much of the content resembles ChatGPT, Gemini, Claude, or Grok writing styles
- Shows **colour-highlighted passages** so you know exactly *which sentences* are likely AI-generated
- Lets you **export a full .txt report** with one click

## ✨ Features

### 🔍 Detection

- **Text & Documents** — Word-level AI pattern analysis via HuggingFace Inference API
- **PDF / DOCX / PPTX** — Full text extraction before analysis
- **Code Files** — Detects AI-assisted code (Python, JS, Java, C++, Go, Rust, and 20+ more)
- **Images** — AI image detection using `umm-maybe/AI-image-detector` model + EXIF metadata scan
- **Videos** — Frame-by-frame analysis with a live timeline (max 2 minutes, max 100 MB)
- **Plain Text** — Paste directly in the browser

### 📊 Results

- **Overall AI %** — Animated donut chart
- **Estimated Model Attribution** — ChatGPT-like / Gemini-like / Claude-like / Other AI-like percentages with animated bars *(heuristic-based)*
- **Highlighted Passages** — Colour-coded by suspected AI model style
- **Video Timeline** — Frame-by-frame AI detection bar
- **One-Click Export** — Download a full `.txt` report

### 🎨 UI / UX

- "Cosmic Candy" theme — Ember Orange + Honey + Deep Charcoal
- Particle network hero background
- Typewriter animation
- Scroll-triggered reveal animations
- Scan overlay with animated progress

## 🚀 Local Setup & Running

### Prerequisites

- Python 3.10+ (`python --version`)
- Git
- A HuggingFace account with an API key (free)

### 1. Clone the Repo

```bash
git clone https://github.com/MohammadSakibAhmad0874/AI-Sentinel.git
cd AI-Sentinel
```

### 2. Backend Setup

```bash
cd backend

# Install dependencies
pip install -r requirements.txt

# Create your .env file
echo HUGGINGFACE_API_KEY=hf_your_key_here > .env

# Start the server
python -m uvicorn main:app --port 8000 --reload
```

> Backend runs at: `http://localhost:8000`
> Swagger docs at: `http://localhost:8000/api/docs`
> Frontend served at: `http://localhost:8000` (auto-served by FastAPI in local mode)

### 3. Open the App

```
http://localhost:8000
```

No separate frontend server needed — FastAPI serves the HTML/CSS/JS directly when running locally.

## 🌐 Production Deployment

### Frontend → [Vercel](https://vercel.com)

1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import `MohammadSakibAhmad0874/AI-Sentinel` from GitHub
3. Leave **Root Directory** as `/` (the `vercel.json` handles routing)
4. Click **Deploy**

> ✅ The `vercel.json` already routes everything through `frontend/`

### Backend → [HuggingFace Spaces](https://huggingface.co/spaces)

1. Go to [huggingface.co/spaces](https://huggingface.co/spaces) → **Create new Space**
2. Name it `ai-sentinel` | SDK: **Docker** | Visibility: **Public**
3. In Space **Settings → Variables and Secrets**, add:
   ```
   HUGGINGFACE_API_KEY = hf_your_key_here
   ```
4. In the Space **Files** tab, upload or link the GitHub repo
5. HF Spaces will build the `Dockerfile` automatically and run on port 7860

**After deployment, update `frontend/js/config.js`:**
```js
const PROD_API_BASE = 'https://YOUR-USERNAME-ai-sentinel.hf.space/api';
```

## 📡 API Reference

Base URL: `https://MohammadSakibAhmad0874-ai-sentinel.hf.space`

### `POST /api/detect`

Detect AI content in a file or text.

**Request** — `multipart/form-data`

| Field | Type | Description |
|-------|------|-------------|
| `file` | File | Any supported file |
| `text` | string | Plain text (alternative to file) |

**Response** — `application/json`

```json
{
  "filename": "essay.docx",
  "file_type": "text",
  "file_size": 24576,
  "ai_percent": 82.3,
  "human_percent": 17.7,
  "is_ai": true,
  "model_attribution": {
    "chatgpt": 45.3,
    "gemini": 22.5,
    "claude": 9.8,
    "other": 4.7
  },
  "chunks": [
    {
      "text": "The implications of...",
      "ai_score": 94.1,
      "is_ai": true
    }
  ],
  "disclaimer": "Model attribution is heuristic-based and not guaranteed.",
  "method": "huggingface_api"
}
```

### `GET /api/health`

```json
{
  "status": "ok",
  "api_key_loaded": true,
  "version": "1.0.0"
}
```

## 🧠 How Detection Works

```
File Upload
    │
    ▼
file_parser.py ──────────► Extracts text / image bytes / video frames
    │
    ├─ Text/Code/Doc ──────► text_detector.py
    │                            │
    │                            ├─ HuggingFace API (Hello-SimpleAI/chatgpt-detector)
    │                            └─ Fallback heuristics (perplexity, burstiness, patterns)
    │
    ├─ Image ──────────────► image_detector.py
    │                            │
    │                            ├─ HuggingFace API (umm-maybe/AI-image-detector)
    │                            └─ EXIF metadata scan (Stable Diffusion, DALL-E markers)
    │
    └─ Video ──────────────► video_detector.py
                                 │
                                 ├─ Duration check (max 2 min)
                                 ├─ Size check (max 100 MB)
                                 └─ Per-frame image analysis via OpenCV
```

### Model Attribution *(Heuristic-Based)*

AI % is split across model styles using writing-pattern analysis:

| Style | Detection Signals |
|-------|-------------------|
| ChatGPT-like | Lists, hedging phrases, "certainly/absolutely" |
| Gemini-like | Academic vocabulary, structured paragraphs |
| Claude-like | Nuanced qualifiers, balanced arguments |
| Other AI | Generic patterns not matching the above |

> ⚠️ **This is NOT true model identification.** It's pattern estimation for guidance only.
